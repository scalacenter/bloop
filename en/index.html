Bloop is a Scala build server.

Compile, test and run Scala fast.

Features

  • Provides fast compile, test and run
  • Has a built-in command-line tool
  • Integrates with most JVM build tools
  • Implements the Build Server Protocol
  • Supports JVM, Scala.js and Scala Native

Why?

  • You edit code and get instant feedback
  • Saves you time and makes you more productive
  • Makes Scala support in build tools easier
  • Adaptable to your own developer workflow

Use it with your favorite build tool

Bloop supports the most popular JVM build tools in the Java and Scala community, with more integrations on their way! Export your build and benefit from faster compile times.

Use it with IDEs

Bloop integrates with IDEs and text editors to provide a short feedback cycle and reliable compiler diagnostics. Use Bloop with IntelliJ and Metals (Visual Studio Code, Sublime Text, vim and Atom).
